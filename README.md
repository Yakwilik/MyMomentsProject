# MyMomentsProject
проект «Моменты жизни», где можно выкладывать воспоминания в формате фотографий с подписями. Возможности комментирования и оценки формируют сообщество и позволяют пользователям активно взаимодействовать друг с другом. В качестве образца реализации рекомендуется использовать Instagram.
